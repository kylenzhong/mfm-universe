# MFM Universe

An interactive intelligence layer over **My First Million** and **Starter Story** — two of the best founder podcasts on the internet.

**Live:** https://mfm-universe.vercel.app

## What's inside

| Page | Purpose |
|---|---|
| `index.html` | Landing page with links to each section |
| `MFM_Episodes.html` | 800+ MFM episodes indexed with key ideas, guests, topic search |
| `MFM_Guest_Profiles.html` | 44 notable MFM guests profiled |
| `Starter_Story.html` | Latest 50 Starter Story episodes mined for apps, revenue, tactics; 8 cross-episode playbooks |
| `MFM_RPG.html` | Pixel-art RPG playthrough of the MFM universe |

## Stack

Pure static HTML/JS/CSS — no build step. Hosted on Vercel.
