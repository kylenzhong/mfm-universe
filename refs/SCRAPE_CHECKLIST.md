# Photo refs checklist — drop into this `refs/` folder

**File naming:** save each photo as `<slug>.jpg` (or .jpeg/.png) into this folder, using the exact slug from the left column.

**Photo selection tips:**
- Clear head-and-shoulders crop works best. Avoid full-body or wide shots.
- Twitter/X avatar = usually perfect (square, face-filling).
- Wikipedia photos, podcast appearance stills, LinkedIn headshots also fine.
- One photo per person. Front-facing or slight 3/4 angle. Eyes visible.
- File extension doesn't matter (.jpg, .jpeg, .png, .webp all work).
